# TextEditor
"Учебный текстовый редактор на Qt для ОС Альт"
Учебный текстовый редактор с расширенными возможностями, разработанный для демонстрации принципов объектно-ориентированного программирования, паттерна MVC и работы с Qt. Проект предназначен для использования в учебных лабораториях с ОС Альт (семейство Linux).

💻 Требования к системе
Для сборки и запуска:
ОС: Альт Рабочая станция / Альт Образование / Simply Linux (или любой дистрибутив Linux с Qt)

Компилятор: GCC (g++) 7.0 или выше

Qt: версия 6.x (модули core, gui, widgets)

Система сборки: qmake (входит в состав Qt)

Дополнительно: make

Проверка установки Qt:
bash
qmake --version
g++ --version
make --version
Если Qt не установлен, установите из репозитория:

bash
sudo apt update
sudo apt install qt6-base-dev qt6-tools-dev g++ make
🔧 Сборка проекта
1. Сборка в Qt Creator (рекомендуемый способ)
Клонируйте репозиторий:

bash
git clone [https://github.com/ваш-логин/TextEditor.git](https://github.com/dankarpen2004-gif/TextEditor/edit/main/README.md)
cd TextEditor
Откройте Qt Creator

Выберите "Открыть проект" (File → Open File or Project)

Выберите файл TextEditor.pro

Настройте комплект сборки (выберите версию Qt и компилятор)

Нажмите "Собрать" (Ctrl+B)

Запустите приложение (Ctrl+R)

2. Сборка из командной строки
bash
# Клонирование репозитория
git clone [https://github.com/ваш-логин/TextEditor.git](https://github.com/dankarpen2004-gif/TextEditor/edit/main/README.md)
cd TextEditor

# Генерация Makefile
qmake TextEditor.pro

# Сборка проекта
make

# Запуск приложения
./TextEditor
Для очистки:

bash
make clean
rm -f TextEditor

Назначение основных классов
Класс	Назначение	Роль в MVC
MainWindow	Главное окно приложения, обработка действий пользователя	Controller
FindDialog	Диалог поиска, отправка сигнала с параметрами	Controller
QTextDocument	Хранение текста (встроенный класс Qt)	Model
QTextEdit	Отображение и редактирование текста (встроенный класс Qt)	View
.ui файлы	Визуальное описание интерфейса	View

🎓 Учебная ценность
Проект демонстрирует следующие концепции:

Концепция	Где реализована
-Инкапсуляция	Классы MainWindow, FindDialog скрывают детали реализации
-Наследование	MainWindow наследует QMainWindow, FindDialog наследует QDialog
-Полиморфизм	Сигналы и слоты Qt, виртуальные методы
-Паттерн MVC	Разделение на модель (QTextDocument), представление (.ui) и контроллер (.cpp)
-Сигналы и слоты	Связь между FindDialog и MainWindow
-Работа с файлами	QFile, QFileDialog, QTextStream
-Тестирование	Встроенные модульные и интеграционные тесты
🧪 Тестирование

cpp
